Basic Operation Instructions

  • Hold select to turn the controller on or off.
  • Hold the following action buttons on power up (Not dpad) to boot into the following modes:
Button Mode
A Nintendo Switch
X XInput
B DInput
Dpad Left NES/SNES
Dpad Down N64
Dpad Right Gamecube

If no USB cable is connected, the controller will boot into wireless mode.

HOJA EFM8UB1 Update Required

This update can only be run from Windows currently.

  1. After flashing the ESP32 firmware, hold L and R while plugging in.
  2. This will enable USB bootloader on the EFM8UB1. Download the zip file linked
  3. Unzip it, and run the included .bat file to update the firmware.
  4. Unplug and replug to finalize the firmware install.
