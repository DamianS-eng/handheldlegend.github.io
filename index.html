Version 1.17.2023.3 - Resolves SNES issues for real this time?

Boot modes are now a thing for wired modes only (for now). Hold a button while plugging in to enable a mode.

Mode is saved and will load on reboot.

Buttons are action buttons, NOT DPAD.

Left button -> Retro Wired Mode (SNES/GC)

Right button -> Nintendo Switch Wired Mode

Up button -> XInput Wired Mode (Steam Compatible)

Down button -> DInput Wired Mode



HOJA EFM8UB1 Update Required

Hold the "EN" pin on your ESP32 while plugging in via USB type C.

This will enable USB bootloader on the EFM8UB1. Download the zip file linked

unzip it, and run the included .bat file to update the firmware.

Unplug and replug to finalize the firmware install.
